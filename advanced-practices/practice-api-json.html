Задача 1: Парсинг JSONЛёгкая

Дана JSON-строка с данными пользователя. Распарсите её и выведите имя и email.

JSON:{"name": "Алекс", "email": "alex@example.com", "age": 25}
Вывод:Алекс, alex@example.com

Задача 3: Список пользователейЛёгкая

Дан JSON-массив пользователей. Выведите имена всех пользователей старше 30 лет.

JSON:[{"name": "Анна", "age": 35}, {"name": "Борис", "age": 28}, {"name": "Виктор", "age": 42}]
Вывод:Анна (35) Виктор (42)
import json

json_string = '[{"name": "Анна", "age": 35}, {"name": "Борис", "age": 28}, {"name": "Виктор", "age": 42}]'
users = json.loads(json_string)

for user in users:
    if user['age'] > 30:
        print(f"{user['name']} ({user['age']})")

Задача 7: POST-запросСредняя

Отправьте POST-запрос для создания нового поста. Выведите ответ сервера.

Вывод:Создан пост с ID: 101 Заголовок: Мой новый пост
import requests

new_post = {
    'title': 'Мой новый пост',
    'body': 'Это текст поста',
    'userId': 1
}

response = requests.post(
    'https://jsonplaceholder.typicode.com/posts',
    json=new_post
)

created = response.json()
print(f"Создан пост с ID: {created['id']}")
print(f"Заголовок: {created['title']}")

Задача 8: Погодное приложениеСложная

Создайте функцию get_weather(city), которая получает погоду из OpenWeatherMap API.

💡 Подсказка

Используйте бесплатный API ключ или тестовые данные. Endpoint: api.openweathermap.org/data/2.5/weather

import requests

API_KEY = "ваш_api_ключ"  # Замените на свой

def get_weather(city):
    url = "https://api.openweathermap.org/data/2.5/weather"
    params = {
        'q': city,
        'appid': API_KEY,
        'units': 'metric',
        'lang': 'ru'
    }
    
    response = requests.get(url, params=params)
    data = response.json()
    
    temp = data['main']['temp']
    feels_like = data['main']['feels_like']
    desc = data['weather'][0]['description']
    
    print(f"Погода в городе {city}:")
    print(f"  Температура: {temp}°C (ощущается как {feels_like}°C)")
    print(f"  Описание: {desc.capitalize()}")

# Пример
get_weather("Moscow")
Задача 9: Агрегация данныхСложная

Получите все комментарии к постам с postId=1 и подсчитайте количество комментариев каждого пользователя.

Вывод:Пользователь 1: 2 комм. Пользователь 3: 1 комм. Пользователь 2: 1 комм. Пользователь 5: 1 комм.
import requests
from collections import Counter

response = requests.get(
    'https://jsonplaceholder.typicode.com/comments',
    params={'postId': 1}
)
comments = response.json()

# Считаем комментарии по пользователям
user_counts = Counter(c['email'] for c in comments)

print("Комментарии по пользователям:")
for email, count in user_counts.items():
    print(f"  {email}: {count} комм.")
Задача 10: JSON to CSVСложная

Получите данные о пользователях и сохраните их в CSV-файл.

import requests
import csv

# Получаем пользователей
response = requests.get('https://jsonplaceholder.typicode.com/users')
users = response.json()

# Сохраняем в CSV
with open('users.csv', 'w', newline='', encoding='utf-8') as f:
    writer = csv.DictWriter(f, fieldnames=['id', 'name', 'email', 'city'])
    writer.writeheader()
    
    for user in users:
        writer.writerow({
            'id': user['id'],
            'name': user['name'],
            'email': user['email'],
            'city': user['address']['city']
        })

print(f"Сохранено {len(users)} пользователей в users.csv")
