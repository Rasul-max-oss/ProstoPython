Задача 1: Чтение CSVЛёгкая

Создайте CSV-файл с данными о продуктах (название, цена, количество) и прочитайте его.

Вывод:Продукт: Яблоки, Цена: 100, Кол-во: 50 Продукт: Молоко, Цена: 80, Кол-во: 30
import csv

# Создаём CSV
with open('products.csv', 'w', newline='', encoding='utf-8') as f:
    writer = csv.writer(f)
    writer.writerow(['Название', 'Цена', 'Количество'])
    writer.writerow(['Яблоки', 100, 50])
    writer.writerow(['Молоко', 80, 30])
    writer.writerow(['Хлеб', 45, 100])

# Читаем CSV
with open('products.csv', 'r', encoding='utf-8') as f:
    reader = csv.reader(f)
    for row in reader:
        if row:  # Пропускаем пустые строки
            print(f"Продукт: {row[0]}, Цена: {row[1]}, Кол-во: {row[2]}")
Задача 2: DictReaderЛёгкая

Прочитайте CSV файл используя DictReader и выведите продукты дороже 50 руб.

Вывод:Яблоки: 100 руб. Молоко: 80 руб.
import csv

with open('products.csv', 'r', encoding='utf-8') as f:
    reader = csv.DictReader(f)
    for row in reader:
        if int(row['Цена']) > 50:
            print(f"{row['Название']}: {row['Цена']} руб.")

Задача 4: Подсчёт суммыСредняя

Прочитайте CSV с продуктами и посчитайте общую стоимость всех товаров (цена × количество).

Вывод:Общая стоимость: 11 900 руб.
import csv

total = 0
with open('products.csv', 'r', encoding='utf-8') as f:
    reader = csv.DictReader(f)
    for row in reader:
        price = int(row['Цена'])
        quantity = int(row['Количество'])
        total += price * quantity

print(f"Общая стоимость: {total:,} руб.")
Задача 5: Фильтрация данныхСредняя

Создайте новый CSV файл только с продуктами, у которых количество больше 40.

import csv

# Читаем исходный файл
products = []
with open('products.csv', 'r', encoding='utf-8') as f:
    reader = csv.DictReader(f)
    for row in reader:
        if int(row['Количество']) > 40:
            products.append(row)

# Записываем отфильтрованные
with open('filtered_products.csv', 'w', newline='', encoding='utf-8') as f:
    fieldnames = ['Название', 'Цена', 'Количество']
    writer = csv.DictWriter(f, fieldnames=fieldnames)
    writer.writeheader()
    writer.writerows(products)

print(f"Сохранено {len(products)} продуктов с количеством > 40")
Задача 6: Группировка по городуСредняя

Дан CSV с сотрудниками. Посчитайте количество сотрудников в каждом городе.

import csv
from collections import Counter

cities = Counter()
with open('employees.csv', 'r', encoding='utf-8') as f:
    reader = csv.DictReader(f)
    for row in reader:
        cities[row['city']] += 1

print("Сотрудников по городам:")
for city, count in cities.items():
    print(f"  {city}: {count} чел.")

Задача 8: Конвертер CSV в JSONСложная

Напишите функцию, которая конвертирует CSV файл в JSON формат.

import csv
import json

def csv_to_json(csv_file, json_file):
    """Конвертирует CSV в JSON"""
    data = []
    with open(csv_file, 'r', encoding='utf-8') as f:
        reader = csv.DictReader(f)
        for row in reader:
            # Преобразуем числа
            for key, value in row.items():
                if value.isdigit():
                    row[key] = int(value)
            data.append(row)
    
    with open(json_file, 'w', encoding='utf-8') as f:
        json.dump(data, f, ensure_ascii=False, indent=2)
    
    print(f"Конвертировано {len(data)} записей")

# Использование
csv_to_json('products.csv', 'products.json')

# Проверка

    print(json.load(f))
