Задача 1: Линейный графикЛёгкая

Постройте линейный график функции y = x² для x от 1 до 10.

import matplotlib.pyplot as plt

x = range(1, 11)
y = [i**2 for i in x]

plt.plot(x, y, marker='o')
plt.xlabel('X')
plt.ylabel('Y = X²')
plt.title('Квадратичная функция')
plt.grid(True, alpha=0.3)
plt.savefig('graph1.png', dpi=100)
plt.show()
print("График сохранён в graph1.png")

Задача 4: Несколько линийСредняя

Постройте на одном графике функции y = x, y = x², y = x³ для x от 1 до 5.

import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y1 = x
y2 = [i**2 for i in x]
y3 = [i**3 for i in x]

plt.figure(figsize=(10, 6))
plt.plot(x, y1, 'o-', label='y = x', color='blue')
plt.plot(x, y2, 's--', label='y = x²', color='green')
plt.plot(x, y3, '^-', label='y = x³', color='red')

plt.xlabel('X')
plt.ylabel('Y')
plt.title('Сравнение функций')
plt.legend()
plt.grid(True, alpha=0.3)
plt.savefig('graph4.png', dpi=100)

Задача 5: ГистограммаСредняя

Постройте гистограмму распределения 100 случайных чисел (нормальное распределение).

💡 Подсказка

Используйте np.random.randn() для нормального распределения.

import matplotlib.pyplot as plt
import numpy as np

np.random.seed(42)
data = np.random.randn(100) * 10 + 50  # mean=50, std=10

plt.figure(figsize=(10, 6))

Задача 7: Динамика продажСредняя

Постройте график динамики продаж за год (12 месяцев) с линией тренда.

import matplotlib.pyplot as plt
import numpy as np

months = ['Янв', 'Фев', 'Мар', 'Апр', 'Май', 'Июн',
          'Июл', 'Авг', 'Сен', 'Окт', 'Ноя', 'Дек']
sales = [100, 120, 115, 140, 160, 180, 175, 190, 210, 230, 250, 280]

plt.figure(figsize=(12, 6))
plt.plot(months, sales, 'o-', linewidth=2, color='green', 
         markersize=8, label='Продажи')

# Линия тренда
z = np.polyfit(range(12), sales, 1)
p = np.poly1d(z)
plt.plot(months, p(range(12)), 'r--', alpha=0.5, label='Тренд')

plt.xlabel('Месяц')
plt.ylabel('Продажи (тыс. руб.)')
plt.title('Динамика продаж за год')
plt.legend()
plt.grid(True, alpha=0.3)
plt.savefig('graph7.png', dpi=100)
plt.show()

Задача 9: Несколько графиковСложная

Создайте фигуру с 4 подграфиками: line, bar, pie, scatter.

import matplotlib.pyplot as plt
import numpy as np

fig, axes = plt.subplots(2, 2, figsize=(12, 10))

# 1. Линейный
x = range(1, 6)
y = [1, 4, 9, 16, 25]
axes[0, 0].plot(x, y, 'o-', color='green')
axes[0, 0].set_title('Линейный')
axes[0, 0].grid(True, alpha=0.3)

# 2. Столбчатый
categories = ['A', 'B', 'C', 'D']
values = [23, 45, 56, 78]
axes[0, 1].bar(categories, values, color='coral')
axes[0, 1].set_title('Столбчатый')

# 3. Круговой
sizes = [30, 25, 25, 20]
axes[1, 0].pie(sizes, labels=['A', 'B', 'C', 'D'], autopct='%1.1f%%')
axes[1, 0].set_title('Круговой')

# 4. Scatter
np.random.seed(42)
axes[1, 1].scatter(np.random.randn(50), np.random.randn(50), 
                   alpha=0.6, color='purple')
axes[1, 1].set_title('Точечный')
axes[1, 1].grid(True, alpha=0.3)

plt.tight_layout()
plt.savefig('graph9.png', dpi=100)
plt.show()
Задача 10: Полный отчётСложная

Создайте полный визуальный отчёт по продажам с сохранением в файл.

import matplotlib.pyplot as plt
import numpy as np

# Данные
months = ['Янв', 'Фев', 'Мар', 'Апр', 'Май', 'Июн']
online = [120, 135, 150, 180, 200, 220]
offline = [80, 75, 70, 65, 60, 55]
total = [o + f for o, f in zip(online, offline)]

# Создаём фигуру
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(14, 6))

# График 1: Линии
ax1.plot(months, online, 'o-', label='Онлайн', 
         color='green', linewidth=2, markersize=8)
ax1.plot(months, offline, 's--', label='Офлайн', 
         color='red', linewidth=2, markersize=8)
ax1.set_xlabel('Месяц')
ax1.set_ylabel('Продажи (тыс. руб.)')
ax1.set_title('Динамика продаж')
ax1.legend()
ax1.grid(True, alpha=0.3)

# График 2: Столбцы
colors = ['skyblue' if t < 250 else 'coral' for t in total]
ax2.bar(months, total, color=colors, edgecolor='black')
ax2.set_xlabel('Месяц')
ax2.set_ylabel('Общая выручка (тыс. руб.)')
ax2.set_title('Общая выручка')

# Значения на столбцах
for i, v in enumerate(total):
    ax2.text(i, v + 5, str(v), ha='center')

plt.tight_layout()
plt.savefig('sales_report.png', dpi=150, bbox_inches='tight')
print("Отчёт сохранён в sales_report.png")
plt.show()
