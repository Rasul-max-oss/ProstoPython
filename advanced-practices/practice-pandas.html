Задача 11: Сводная таблицаСложная

Создайте сводную таблицу: строки — категории, столбцы — месяцы, значения — сумма продаж.

import pandas as pd
import numpy as np

np.random.seed(42)
data = {
    'Месяц': np.random.choice(['Янв', 'Фев', 'Мар'], 50),
    'Категория': np.random.choice(['Электроника', 'Одежда', 'Дом'], 50),
    'Продажи': np.random.randint(1000, 50000, 50)
}
df = pd.DataFrame(data)

pivot = df.pivot_table(
    values='Продажи',
    index='Категория',
    columns='Месяц',
    aggfunc='sum',
    fill_value=0
)

print("Сводная таблица продаж:")
print(pivot)

Задача 13: Анализ временных рядовСложная

Создайте данные о ежедневных продажах за месяц. Выведите день с максимальной выручкой и среднюю выручку по неделям.

import pandas as pd
import numpy as np

# Данные за 30 дней
dates = pd.date_range('2024-01-01', periods=30, freq='D')
sales = np.random.randint(10000, 100000, 30)

df = pd.DataFrame({'Дата': dates, 'Выручка': sales})
df['Неделя'] = df['Дата'].dt.isocalendar().week

# День с максимальной выручкой
max_day = df.loc[df['Выручка'].idxmax()]
print(f"Максимальная выручка: {max_day['Выручка']:,} руб.")
print(f"Дата: {max_day['Дата'].strftime('%d.%m.%Y')}")

# Средняя по неделям
weekly = df.groupby('Неделя')['Выручка'].mean()
print("\nСредняя выручка по неделям:")
print(weekly)
Задача 14: Полный анализСложная

Создайте отчёт по продажам: общая выручка, топ категории, динамика по месяцам, экспорт в Excel.

import pandas as pd
import numpy as np

# Генерация данных
np.random.seed(42)
n = 200
data = {
    'Дата': pd.date_range('2024-01-01', periods=n, freq='D'),
    'Категория': np.random.choice(['Электроника', 'Одежда', 'Дом', 'Спорт'], n),
    'Товар': np.random.choice(['Товар'+str(i) for i in range(1, 21)], n),
    'Продано': np.random.randint(1, 20, n),
    'Цена': np.random.randint(500, 50000, n)
}
df = pd.DataFrame(data)
df['Выручка'] = df['Продано'] * df['Цена']
df['Месяц'] = df['Дата'].dt.month

print("=" * 50)
print("ОТЧЁТ ПО ПРОДАЖАМ")
print("=" * 50)

# Общая выручка
total = df['Выручка'].sum()
print(f"\nОбщая выручка: {total:,.0f} руб.")

# Топ категория
top_cat = df.groupby('Категория')['Выручка'].sum().idxmax()
print(f"Топ категория: {top_cat}")

# Динамика по месяцам
print("\nДинамика по месяцам:")
monthly = df.groupby('Месяц')['Выручка'].sum()
for month, rev in monthly.items():
    print(f"  Месяц {month}: {rev:,.0f} руб.")

# Экспорт в Excel
with pd.ExcelWriter('sales_report.xlsx') as writer:
    df.to_excel(writer, sheet_name='Данные', index=False)
    monthly.to_excel(writer, sheet_name='По месяцам')
    pd.pivot_table(df, values='Выручка', index='Категория', 
                   columns='Месяц', aggfunc='sum').to_excel(
                       writer, sheet_name='Сводная')

print("\nОтчёт сохранён в sales_report.xlsx")
