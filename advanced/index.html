Все темы для уверенного владения Python: от объектно-ориентированного программирования до анализа данных и работы с базами данных.

  • Урок 2: Работа с API и JSON
  • Урок 3: Обработка CSV таблиц

  • Урок 5: Анализ данных — Pandas и Numpy
  • Урок 6: Сетевые запросы (requests)
  • Урок 7: Базы данных SQLite
