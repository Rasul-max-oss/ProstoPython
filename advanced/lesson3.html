Обработка CSV таблиц

CSV (Comma-Separated Values) — универсальный формат для хранения табличных данных. Научимся читать, записывать и обрабатывать CSV-файлы в Python.

Что такое CSV

CSV — это текстовый файл, где данные разделены запятыми (или другими разделителями). Каждая строка — это запись таблицы.

Пример CSV-файла

Так выглядит файл employees.csv в текстовом редакторе:

employees.csv Исходные данные 
id,name,age,department,salary
1,Иван Петров,28,IT,75000
2,Анна Смирнова,34,HR,65000
3,Дмитрий Козлов,45,Finance,95000
4,Ольга Новикова,29,IT,80000
5,Сергей Морозов,38,Marketing,70000

Первая строка — заголовки столбцов. Остальные строки — данные сотрудников.

Чтение CSV файла

В Python есть встроенный модуль csv для работы с такими файлами.

main.py Пример 1: Чтение CSV 
import csv

# Открываем CSV файл
with open('employees.csv', 'r', encoding='utf-8') as file:
    # Создаём читатель CSV
    reader = csv.reader(file)
    
    # Получаем заголовок
    header = next(reader)
    print("Заголовки:", header)
    print()
    
    # Читаем остальные строки
    for row in reader:
        print(row)

['1', 'Иван Петров', '28', 'IT', '75000']

next(reader) — возвращает следующую строку из файла. Первый вызов получает заголовок.

Чтение как словарь (DictReader)

Удобнее работать с данными, когда каждая строка — словарь с ключами-заголовками.

main.py Пример 2: CSV как словари 
import csv

with open('employees.csv', 'r', encoding='utf-8') as file:
    # DictReader автоматически использует первую строку как ключи
    reader = csv.DictReader(file)
    
    for row in reader:
        print(f"Сотрудник: {row['name']}")
        print(f"  Возраст: {row['age']} лет")
        print(f"  Отдел: {row['department']}")
        print(f"  Зарплата: {row['salary']} руб.")
        print("-" * 30)

Анализ данных из CSV

Теперь научимся выполнять вычисления с данными из файла.

main.py Пример 3: Анализ данных 
import csv

employees = []

# Читаем все данные
with open('employees.csv', 'r', encoding='utf-8') as file:
    reader = csv.DictReader(file)
    for row in reader:
        employees.append(row)

# 1. Общее количество сотрудников
print(f"Всего сотрудников: {len(employees)}")

# 2. Средняя зарплата
salaries = [int(emp['salary']) for emp in employees]
avg_salary = sum(salaries) / len(salaries)
print(f"Средняя зарплата: {avg_salary:,.0f} руб.")

# 3. Сотрудник с максимальной зарплатой
max_emp = max(employees, key=lambda x: int(x['salary']))
print(f"\nСамая высокая зарплата у: {max_emp['name']}")
print(f"Зарплата: {max_emp['salary']} руб.")

# 4. Средний возраст по отделам
departments = {}
for emp in employees:
    dept = emp['department']
    if dept not in departments:
        departments[dept] = []
    departments[dept].append(int(emp['age']))

print("\nСредний возраст по отделам:")
for dept, ages in departments.items():
    avg_age = sum(ages) / len(ages)
    print(f"  {dept}: {avg_age:.1f} лет")

Вывод программы:

Терминал Результат 
Всего сотрудников: 5
Средняя зарплата: 77,000 руб.

Самая высокая зарплата у: Дмитрий Козлов
Зарплата: 95000 руб.

Средний возраст по отделам:
  IT: 28.5 лет
  HR: 34.0 лет
  Finance: 45.0 лет
  Marketing: 38.0 лет

Запись данных в CSV

Создадим новый CSV файл с результатами анализа.

main.py Пример 4: Запись CSV 
import csv

# Данные для записи
new_employees = [
    {'id': 6, 'name': 'Павел Лебедев', 'age': 31, 'department': 'IT', 'salary': 85000},
    {'id': 7, 'name': 'Елена Соколова', 'age': 27, 'department': 'HR', 'salary': 60000},
    {'id': 8, 'name': 'Максим Павлов', 'age': 42, 'department': 'Finance', 'salary': 100000}
]

# Заголовки
fieldnames = ['id', 'name', 'age', 'department', 'salary']

# Записываем в файл
with open('employees_updated.csv', 'w', encoding='utf-8', newline='') as file:
    writer = csv.DictWriter(file, fieldnames=fieldnames)
    
    # Записываем заголовок
    writer.writeheader()
    
    # Записываем данные
    writer.writerows(new_employees)

print("Данные успешно сохранены в employees_updated.csv")

# Проверка - читаем обратно
print("\nСодержимое файла:")
with open('employees_updated.csv', 'r', encoding='utf-8') as file:
    print(file.read())

Вывод программы:

Терминал Результат 
Данные успешно сохранены в employees_updated.csv

Содержимое файла:

7,Елена Соколова,27,HR,60000
8,Максим Павлов,42,Finance,100000
newline='' — важный параметр для Windows, предотвращает пустые строки в CSV.

Добавление данных в существующий файл

Часто нужно не перезаписать, а дополнить файл.

main.py Пример 5: Добавление записей 
import csv

# Новые сотрудники для добавления
new_employees = [
    {'id': 9, 'name': 'Кристина Орлова', 'age': 25, 'department': 'Marketing', 'salary': 55000},
    {'id': 10, 'name': 'Артем Жуков', 'age': 36, 'department': 'IT', 'salary': 90000}
]

fieldnames = ['id', 'name', 'age', 'department', 'salary']

# Открываем файл на дозапись
with open('employees_updated.csv', 'a', encoding='utf-8', newline='') as file:
    writer = csv.DictWriter(file, fieldnames=fieldnames)
    writer.writerows(new_employees)

print("Добавлено 2 новых сотрудника")

# Читаем итоговый файл
with open('employees_updated.csv', 'r', encoding='utf-8') as file:
    reader = csv.DictReader(file)
    for row in reader:
        print(f"{row['id']}. {row['name']} - {row['department']}")

Другие разделители

CSV может использовать не только запятые, но и другие разделители: точку с запятой, табуляцию, вертикальную черту.

data_semicolon.csv Файл с точкой с запятой 
name;city;score
Алексей;Москва;85
Мария;СПб;92
Николай;Казань;78
main.py Пример 6: Чтение с разделителем 
import csv

with open('data_semicolon.csv', 'r', encoding='utf-8') as file:
    # Указываем разделитель точка с запятой
    reader = csv.reader(file, delimiter=';')
    
    for row in reader:
        print(row)

['Алексей', 'Москва', '85']

['Николай', 'Казань', '78']

analysis.py Пример 7: Анализ продаж 
import csv
from datetime import datetime

# Читаем данные
sales = []
with open('sales.csv', 'r', encoding='utf-8') as file:
    reader = csv.DictReader(file)
    for row in reader:
        row['quantity'] = int(row['quantity'])
        row['price'] = int(row['price'])
        row['total'] = row['quantity'] * row['price']
        sales.append(row)

print("=" * 50)
print("ОТЧЁТ ПО ПРОДАЖАМ")
print("=" * 50)

# 1. Общая выручка
total_revenue = sum(s['total'] for s in sales)
print(f"\nОбщая выручка: {total_revenue:,.0f} руб.")

# 2. Продано товаров
total_items = sum(s['quantity'] for s in sales)
print(f"Всего продано товаров: {total_items} шт.")

# 3. Выручка по категориям
print("\nВыручка по категориям:")
categories = {}
for s in sales:
    cat = s['category']
    if cat not in categories:
        categories[cat] = 0
    categories[cat] += s['total']

for cat, revenue in categories.items():
    print(f"  {cat}: {revenue:,.0f} руб.")

# 4. Топ-3 товара по выручке
print("\nТоп-3 товара по выручке:")
sorted_sales = sorted(sales, key=lambda x: x['total'], reverse=True)
for i, s in enumerate(sorted_sales[:3], 1):
    print(f"  {i}. {s['product']}: {s['total']:,.0f} руб. ({s['quantity']} шт.)")

# 5. Средний чек
avg_check = total_revenue / len(sales)
print(f"\nСредний чек: {avg_check:,.0f} руб.")

Экспорт отчёта в CSV

Сохраним результаты анализа в новый файл.

report.py Пример 8: Создание отчёта 
import csv

# Читаем исходные данные
sales = []
with open('sales.csv', 'r', encoding='utf-8') as file:
    reader = csv.DictReader(file)
    for row in reader:
        row['quantity'] = int(row['quantity'])
        row['price'] = int(row['price'])
        sales.append(row)

# Группируем по товарам
product_stats = {}
for s in sales:
    product = s['product']
    if product not in product_stats:
        product_stats[product] = {
            'category': s['category'],
            'quantity': 0,
            'revenue': 0
        }
    product_stats[product]['quantity'] += s['quantity']
    product_stats[product]['revenue'] += s['quantity'] * s['price']

# Записываем отчёт
with open('product_report.csv', 'w', encoding='utf-8', newline='') as file:
    fieldnames = ['product', 'category', 'total_quantity', 'total_revenue']
    writer = csv.DictWriter(file, fieldnames=fieldnames)
    
    writer.writeheader()
    for product, stats in product_stats.items():
        writer.writerow({
            'product': product,
            'category': stats['category'],
            'total_quantity': stats['quantity'],
            'total_revenue': stats['revenue']
        })

print("Отчёт сохранён в product_report.csv")

# Показываем содержимое
print("\nСодержимое отчёта:")
with open('product_report.csv', 'r', encoding='utf-8') as file:
    print(file.read())
