Визуализация данных в Matplotlib

Matplotlib — самая популярная библиотека для построения графиков в Python. Научимся создавать линейные графики, столбчатые диаграммы, круговые диаграммы и многое другое.

Установка и начало работы

Matplotlib не входит в стандартную библиотеку Python, поэтому её нужно установить.

Терминал Установка 
pip install matplotlib

Основные компоненты

Figure —整个 фигура (окно или файл с графиком)
Axes — область с осями координат (сам график)
Plot — линия или точки на графике

Простой линейный график

Начнём с базового примера — график функции y = x².

main.py Пример 1: Первый график 
import matplotlib.pyplot as plt

# Данные для графика
x = [1, 2, 3, 4, 5]
y = [1, 4, 9, 16, 25]  # x в квадрате

# Создаём график
plt.plot(x, y)

# Добавляем подписи
plt.xlabel('Ось X')
plt.ylabel('Ось Y')
plt.title('Простой линейный график')

# Показываем график
plt.show()

Что получится:

  9 |       *

  4 |   *

  1 |*

Линия соединяет точки (1,1), (2,4), (3,9), (4,16), (5,25)
plt.plot() — создаёт линейный график
plt.show() — отображает график в окне
plt.savefig('graph.png') — сохраняет график в файл

Настройка стиля графика

Можно изменить цвет, толщину линии, добавить маркеры точек.

main.py Пример 2: Стилизация 
import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [1, 4, 9, 16, 25]

# 'o-' означает: точки (o) соединённые линией (-)
# color='red' — цвет линии
# linewidth=2 — толщина линии
# markersize=8 — размер маркеров
plt.plot(x, y, 'o-', color='green', linewidth=2, markersize=8)

# Сетка для удобства
plt.grid(True, alpha=0.3)

plt.xlabel('Значения X')
plt.ylabel('Значения Y')
plt.title('График с настройкой стиля')

plt.show()

Форматы линии

'-' — сплошная линия
'--' — пунктир
'o' — только точки
's' — квадраты
'^' — треугольники
'*' — звёздочки

Несколько линий на одном графике

Часто нужно сравнить несколько наборов данных.

main.py Пример 3: Несколько линий 
import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y1 = [1, 4, 9, 16, 25]    # x²
y2 = [1, 8, 27, 64, 125]  # x³

# Строим два графика на одних осях
plt.plot(x, y1, 'o-', label='x²', color='blue')
plt.plot(x, y2, 's--', label='x³', color='red')

# Добавляем легенду
plt.legend()

plt.grid(True, alpha=0.3)
plt.xlabel('X')
plt.ylabel('Y')
plt.title('Сравнение двух функций')

plt.show()

Что получится:

    |           □

    |   *       *
  9 | *       *
    |
  1 |*___*___*___*___*

    1   2   3   4   5

Синяя линия с точками (o) — x²
Красная пунктирная с квадратами (□) — x³

Столбчатая диаграмма

Столбчатые диаграммы (bar charts) используются для сравнения категорий.

main.py Пример 4: Столбчатая диаграмма 
import matplotlib.pyplot as plt

# Категории и значения
categories = ['Январь', 'Февраль', 'Март', 'Апрель', 'Май']
sales = [120, 150, 95, 180, 210]

# Создаём столбчатую диаграмму
plt.bar(categories, sales, color='skyblue', edgecolor='blue')

# Добавляем значения на столбцы
for i, v in enumerate(sales):
    plt.text(i, v + 5, str(v), ha='center', fontsize=10)

plt.xlabel('Месяц')
plt.ylabel('Продажи (тыс. руб.)')
plt.title('Продажи по месяцам')

# Поворачиваем подписи месяцев
plt.xticks(rotation=45)

plt.tight_layout()  # Автоматически подгоняет размеры
plt.show()

Горизонтальная столбчатая диаграмма

Удобна, когда названия категорий длинные.

main.py Пример 5: Горизонтальные столбцы 
import matplotlib.pyplot as plt

# Данные
products = ['Ноутбуки', 'Смартфоны', 'Планшеты', 'Наушники', 'Часы']
revenue = [450, 380, 120, 95, 75]  # в тысячах рублей

# Горизонтальная диаграмма (barh)
plt.barh(products, revenue, color='coral')

# Добавляем значения
for i, v in enumerate(revenue):
    plt.text(v + 10, i, f'{v} тыс.', va='center', fontsize=10)

plt.xlabel('Выручка (тыс. руб.)')
plt.title('Продажи по категориям товаров')

plt.tight_layout()
plt.show()

Круговая диаграмма

Показывает доли целого. Идеально для отображения процентов.

main.py Пример 6: Круговая диаграмма 
import matplotlib.pyplot as plt

# Данные
categories = ['Продукты', 'Транспорт', 'Развлечения', 'Коммуналка', 'Другое']
expenses = [15000, 5000, 8000, 6000, 4000]

# Цвета для каждого сектора
colors = ['#ff9999', '#66b3ff', '#99ff99', '#ffcc99', '#c2c2f0']

# Создаём круговую диаграмму
plt.figure(figsize=(8, 8))  # Размер фигуры
plt.pie(
    expenses,
    labels=categories,
    colors=colors,
    autopct='%1.1f%%',  # Формат процентов
    startangle=90,       # Начальный угол
    explode=(0.1, 0, 0, 0, 0),  # Выделяем первый сектор
    shadow=True
)

plt.title('Расходы семьи за месяц')
plt.axis('equal')  # Круг будет кругом, а не овалом
plt.show()

Что получится:

Результат Визуализация

        │    ╱40.0%╲  │  Продукты (выделен)
        │   ╱       ╲ │  15000 руб.
        │  │  Развл. │ │
        │  │ 21.1%  │ │

Всего расходов: 38 000 руб.
explode — "вытягивает" сектор из центра (кортеж с 0 и 0.1)
autopct — автоматически добавляет проценты
startangle — угол начала первого сектора

Гистограмма

Гистограмма показывает распределение данных по интервалам.

main.py Пример 7: Гистограмма 
import matplotlib.pyplot as plt
import random

# Генерируем случайные данные (оценки студентов)
random.seed(42)
grades = [random.randint(2, 5) for _ in range(100)]

# Создаём гистограмму
plt.figure(figsize=(10, 6))
plt.hist(grades, bins=4, color='steelblue', edgecolor='black', alpha=0.7)

# bins=4 означает 4 интервала (2, 3, 4, 5)
plt.xlabel('Оценка')
plt.ylabel('Количество студентов')
plt.title('Распределение оценок за экзамен')

# Добавляем сетку
plt.grid(axis='y', alpha=0.3)

plt.show()

# Печатаем статистику
from collections import Counter
counts = Counter(grades)
print("Статистика оценок:")
for grade in sorted(counts.keys()):
    print(f"  Оценка {grade}: {counts[grade]} студентов")

Вывод программы:

Терминал Результат 
Статистика оценок:
  Оценка 2: 8 студентов
  Оценка 3: 22 студента
  Оценка 4: 38 студентов
  Оценка 5: 32 студента

На гистограмме видно, что большинство получили 4 и 5

Точечная диаграмма (Scatter plot)

Показывает взаимосвязь между двумя переменными.

main.py Пример 8: Точечная диаграмма 
import matplotlib.pyplot as plt

# Данные: возраст и зарплата
age = [22, 25, 28, 32, 35, 38, 42, 45, 48, 52]
salary = [45, 52, 58, 65, 72, 80, 88, 95, 100, 110]  # в тысячах

plt.figure(figsize=(10, 6))
plt.scatter(age, salary, color='purple', s=100, alpha=0.6)

# Добавляем подписи к точкам
for i, txt in enumerate(age):
    plt.annotate(f'{age[i]} лет', (age[i], salary[i]), 
                 xytext=(5, 5), textcoords='offset points',
                 fontsize=8, alpha=0.7)

plt.xlabel('Возраст (лет)')
plt.ylabel('Зарплата (тыс. руб.)')
plt.title('Зависимость зарплаты от возраста')
plt.grid(True, alpha=0.3)

plt.show()

Сохранение графика в файл

Графики можно сохранять для отчётов и презентаций.

main.py Пример 9: Сохранение 
import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [1, 4, 9, 16, 25]

plt.figure(figsize=(10, 6))
plt.plot(x, y, 'o-', linewidth=2, color='green')
plt.grid(True, alpha=0.3)
plt.xlabel('X')
plt.ylabel('Y')
plt.title('График функции y = x²')

# Сохраняем в файл
# dpi=300 — высокое качество для печати
# bbox_inches='tight' — обрезает лишние поля
plt.savefig('graph.png', dpi=300, bbox_inches='tight')
plt.savefig('graph.pdf', bbox_inches='tight')  # PDF для печати

print("Графики сохранены:")
print("  - graph.png (300 DPI)")
print("  - graph.pdf (векторный)")

# После savefig() можно показать график
plt.show()

Вывод программы:

Терминал Результат 
Графики сохранены:
  - graph.png (300 DPI)
  - graph.pdf (векторный)
Форматы файлов:
.png — растровый, для веба
.pdf — векторный, для печати
.svg — векторный, для веба
.jpg — растровый, меньше размер

Практический проект: Визуализация продаж

Создадим полный отчёт с несколькими графиками.

report.py Пример 10: Полный отчёт 
import matplotlib.pyplot as plt

# Данные о продажах
months = ['Янв', 'Фев', 'Мар', 'Апр', 'Май', 'Июн']
online_sales = [120, 135, 150, 180, 200, 220]
offline_sales = [80, 75, 70, 65, 60, 55]

# Создаём фигуру с двумя графиками
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(14, 6))

# График 1: Линейный график сравнения
ax1.plot(months, online_sales, 'o-', label='Онлайн', 
         color='green', linewidth=2, markersize=8)
ax1.plot(months, offline_sales, 's--', label='Офлайн', 
         color='red', linewidth=2, markersize=8)
ax1.set_xlabel('Месяц')
ax1.set_ylabel('Продажи (тыс. руб.)')
ax1.set_title('Динамика продаж')
ax1.legend()
ax1.grid(True, alpha=0.3)

# График 2: Столбчатая диаграмма общей выручки
total = [o + f for o, f in zip(online_sales, offline_sales)]
colors = ['skyblue' if t < 200 else 'coral' for t in total]
ax2.bar(months, total, color=colors, edgecolor='black')
ax2.set_xlabel('Месяц')
ax2.set_ylabel('Общая выручка (тыс. руб.)')
ax2.set_title('Общая выручка по месяцам')

# Добавляем значения на столбцы
for i, v in enumerate(total):
    ax2.text(i, v + 5, str(v), ha='center', fontsize=10)

plt.tight_layout()
plt.savefig('sales_report.png', dpi=150, bbox_inches='tight')
print("Отчёт сохранён в sales_report.png")
plt.show()
