Установка VS Code

Прежде чем писать код, нужно настроить рабочее место. VS Code — самый популярный редактор кода в мире.

Что такое VS Code

Visual Studio Code (VS Code) — это бесплатный редактор кода от Microsoft. Он поддерживает Python и сотни других языков программирования.

Почему VS Code:
  • Бесплатный и открытый
  • Работает на Windows, macOS и Linux
  • Огромное количество расширений
  • Встроенный терминал и отладчик

Шаг 1: Скачивание

Перейди на официальный сайт code.visualstudio.com и нажми кнопку Download для своей операционной системы.

Шаг 2: Установка

Запусти скачанный файл и следуй инструкциям установщика. Рекомендуется оставить все настройки по умолчанию.

Рекомендуемые настройки
✓ Добавить в PATH
✓ Создать ярлык на рабочем столе
✓ Открыть через VS Code (контекстное меню)

Шаг 3: Установка расширения Python

После установки VS Code нужно добавить поддержку Python:

  1. Открой VS Code
  2. Нажми на иконку расширений слева (квадраты) или Ctrl+Shift+X
  3. В поиске введи "Python"
  4. Установи расширение от Microsoft

Проверка установки

Создай новый файл с расширением .py и напиши простой код:

hello.py
print("Привет, мир!")
print("VS Code готов к работе!")

Запусти код нажатием F5 или через меню Run → Start Debugging.

Совет: Установи также расширение "Pylance" — оно улучшает автодополнение и подсветку кода на Python.
