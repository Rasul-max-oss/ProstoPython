Обработка исключений

Исключения — это ошибки во время выполнения. Научимся обрабатывать их gracefully, чтобы программа не завершалась аварийно.

1. Что такое исключения

Исключение возникает, когда Python встречает ошибку. Без обработки программа завершается.

exception_example.py
# Деление на ноль
result = 10 / 0  # ZeroDivisionError

# Преобразование строки в число
num = int("abc")  # ValueError

# Доступ к несуществующему ключу
data = {"name": "Alex"}
print(data["age"])  # KeyError

2. Конструкция try-except

Блок try-except позволяет перехватывать и обрабатывать исключения.

try_except.py
try:
    number = int(input("Введите число: "))
    result = 10 / number
    print(f"Результат: {result}")
except ValueError:
    print("Ошибка: нужно ввести число!")
except ZeroDivisionError:
    print("Ошибка: деление на ноль!")
Важно: Всегда перехватывай конкретные типы исключений, а не все подряд. Это поможет найти ошибки в коде.

4. Блок else и finally

else выполняется, если не было исключения. finally выполняется всегда.

else_finally.py
try:
    number = int(input("Введите число: "))
    result = 10 / number
except ValueError:
    print("Нужно ввести число!")
except ZeroDivisionError:
    print("На ноль делить нельзя!")
else:
    print(f"Результат: {result}")
finally:
    print("Выполняется всегда")

5. Практические примеры

Пример 1

Безопасный ввод

Запрос числа до успешного ввода

def get_number():
    while True:
        try:
            num = float(input("Введите число: "))
            return num
        except ValueError:
            print("Ошибка! Введите корректное число.")

age = get_number()
print(f"Вы ввели: {age}")
Пример 2

Чтение файла

Обработка отсутствия файла

def read_file(filename):
    try:
        with open(filename, "r", encoding="utf-8") as f:
            return f.read()
    except FileNotFoundError:
        print(f"Файл {filename} не найден")
        return None
    except PermissionError:
        print(f"Нет доступа к файлу {filename}")
        return None

content = read_file("data.txt")
Пример 3

Калькулятор с защитой

Полная обработка ошибок ввода

def calculator():
    try:
        num1 = float(input("Первое число: "))
        num2 = float(input("Второе число: "))
        operation = input("Операция (+, -, *, /): ")
        
        if operation == "+":
            result = num1 + num2
        elif operation == "-":
            result = num1 - num2
        elif operation == "*":
            result = num1 * num2
        elif operation == "/":
            result = num1 / num2
        else:
            print("Неверная операция")
            return
        
        print(f"Результат: {result}")
    except ValueError:
        print("Введите корректные числа!")
    except ZeroDivisionError:
        print("На ноль делить нельзя!")

calculator()

6. Генерация исключений

Можно создавать собственные ошибки с помощью raise.

raise_example.py
def set_age(age):
    if age < 0:
        raise ValueError("Возраст не может быть отрицательным")
    if age > 150:
        raise ValueError("Неверный возраст")
    print(f"Возраст: {age}")

# Вызов
set_age(-5)  # Вызовет ValueError
Логирование: Для записи ошибок в лог используй модуль logging. Это лучше, чем print, для production-кода.
