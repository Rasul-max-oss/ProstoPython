Модули и импорт

Модули позволяют разделять код на файлы и использовать готовые библиотеки. Изучим импорт и создание собственных модулей.

1. Импорт модулей

Python имеет богатую стандартную библиотеку модулей. Для использования модуля нужно его импортировать.

import_basic.py
# Импорт всего модуля
import math

print(math.sqrt(16))  # 4.0
print(math.pi)        # 3.14159...

# Импорт с псевдонимом
import datetime as dt

print(dt.datetime.now())

2. Конкретные импорты

Можно импортировать только нужные функции из модуля.

from_import.py
# Импорт конкретных функций
from math import sqrt, pi

print(sqrt(16))  # 4.0 - не нужно math.
print(pi)        # 3.14159...

# Импорт всех имён из модуля (не рекомендуется)
from math import *

print(sqrt(25))  # 5.0
Лучшая практика: Импортируй конкретные имена или используй import module. Избегай from module import *.

4. Практические примеры

Пример 1

Генератор паролей

Использование модуля random

import random
import string

def generate_password(length=12):
    chars = string.ascii_letters + string.digits + string.punctuation
    password = ''.join(random.choice(chars) for _ in range(length))
    return password

print(generate_password(16))
Пример 2

Работа с датой

Использование модуля datetime

from datetime import datetime, timedelta

# Текущая дата
now = datetime.now()
print(f"Сегодня: {now.strftime('%d.%m.%Y')}")
print(f"Время: {now.strftime('%H:%M')}")

# Дата через неделю
next_week = now + timedelta(days=7)
print(f"Через неделю: {next_week.strftime('%d.%m.%Y')}")
Пример 3

Калькулятор с math

Математические вычисления

import math

def circle_area(radius):
    """Площадь круга"""
    return math.pi * radius ** 2

def hypotenuse(a, b):
    """Гипотенуза треугольника"""
    return math.hypot(a, b)

print(f"Площадь круга R=5: {circle_area(5):.2f}")
print(f"Гипотенуза 3,4: {hypotenuse(3, 4)}")

5. Создание собственного модуля

Любой Python-файл может быть модулем. Создай файл mymodule.py и импортируй его.

mymodule.py
# Файл mymodule.py

def greet(name):
    return f"Привет, {name}!"

def add(a, b):
    return a + b

VERSION = "1.0"
main.py
# Файл main.py
import mymodule

print(mymodule.greet("Алекс"))  # Привет, Алекс!
print(mymodule.add(5, 3))       # 8
print(mymodule.VERSION)         # 1.0

6. Модуль __name__

Конструкция if __name__ == "__main__" позволяет запускать код только при прямом запуске файла.

module_main.py
# Файл calculator.py

def add(a, b):
    return a + b

if __name__ == "__main__":
    # Этот код выполнится только при прямом запуске
    print("Тестирование:")
    print(add(2, 3))
    
# При импорте этот код не выполнится
Зачем это нужно: Позволяет использовать файл и как модуль для импорта, и как самостоятельную программу.
Установка сторонних модулей: Используй pip install package_name для установки библиотек из PyPI (Python Package Index).
