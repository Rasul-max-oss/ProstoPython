Переменные и ввод данных

Справочник по работе с данными в Python: от создания первой переменной до получения ответов от пользователя.

1. Переменные (Variables)

Переменная — это способ сохранить данные под определенным именем. В Python не нужно указывать тип данных заранее.

variables.py
# Имя = Значение
user_name = "Петя"  # Текст (string)
score = 100          # Число (integer)

print(user_name)     # Выведет: Петя
variables.py
# Имя = Значение
name = "Петя"  # Текст (string)
age = 20          # Число (integer)
city = "Москва"  # Текст (string)
balance = 150.75  # Число с плавающей точкой (float)
is_active = True  # Логический тип (boolean)

print(f"Имя: {name}, Возраст: {age}, Город: {city}, Баланс: {balance}, Активен: {is_active}")
# Выведет: Имя: Петя, Возраст: 20, Город: Москва, Баланс: 150.75, Активен: True
💡 Имена: Используй snake_case (маленькие буквы и нижнее подчеркивание). Нельзя начинать имя с цифры.

2. Основные типы данных

Тип Описание Пример
str Строки (текст в кавычках) "Hello"
int Целые числа 25, -5
float Числа с плавающей точкой 3.14, 0.5
bool Логический тип (Истина/Ложь) True, False

3. Ввод данных: input()

Функция input() позволяет программе "спрашивать" данные у пользователя.

input.py
name = input("Как тебя зовут? ")
print("Привет, " + name)

⚠️ Важно: input() всегда принимает данные как строку (str), даже если введено число.

Практика: Работа с вводом

Давай разберем три типичные ситуации, где тебе понадобится ввод данных.

1. Сбор текстовых данных (str)

Используется для имен, названий городов или паролей. Никаких дополнительных функций не нужно.

profile.py
city = input("В каком городе ты живешь? ")
hobby = input("Твое любимое хобби? ")

print("Круто! Я тоже люблю " + hobby + ", особенно в городе " + city)

2. Числовые расчеты (int + input)

Если ты хочешь умножать, складывать или вычитать введенные данные, обязательно используй int().

salary_calc.py
# Считаем бюджет на неделю
daily_budget = int(input("Сколько денег ты тратишь в день? "))
weekly_total = daily_budget * 7

print("За неделю тебе понадобится: " + str(weekly_total) + " рублей")
# Обрати внимание: при выводе числа с текстом мы превратили его обратно в str()

3. Работа с ценами и весом (float + input)

Если пользователь может ввести число с точкой (например, 2.5), используй float().

shop.py
price = float(input("Сколько стоит 1 кг яблок? "))
weight = float(input("Сколько кг ты хочешь купить? "))

total = price * weight
print("К оплате: " + str(total))
⚡️ Быстрая проверка:

Если ты забыл поставить int() и попробовал сложить два ввода "10" и "5", Python не выдаст ошибку, он просто склеит их и получится 105. Если видишь такой странный результат — значит, ты забыл про преобразование типов!

4. Преобразование типов (Casting)

Если нужно проводить математические операции с вводом пользователя, используй функции преобразования:

casting.py
# Превращаем текст в целое число
age = int(input("Введите ваш возраст: "))
next_year = age + 1

# Превращаем число в строку (для склейки с текстом)
print("В следующем году вам будет " + str(next_year))
Основные функции:
  • int() — в целое число
  • float() — в дробное число
  • str() — в строку
