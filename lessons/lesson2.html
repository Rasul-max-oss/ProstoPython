Переменные и типы данных

Переменные хранят данные в программе. Изучим основные типы данных и работу с вводом от пользователя.

1. Переменные

Переменная — это именованное место в памяти для хранения данных. В Python не нужно указывать тип переменной заранее.

# Создание переменных name = "Алекс" age = 25 height = 175.5 is_student = False # Вывод значений print(name) print(age) print(height) print(is_student)
Правила именования:
  • Используй snake_case: user_name, total_price
  • Нельзя начинать с цифры: name1 можно, 1name нельзя
  • Имена чувствительны к регистру: Name и name — разные переменные

2. Основные типы данных

Тип Описание Пример
str Строки (текст) "Hello", "Москва"
int Целые числа 25, -10, 0
float Дробные числа 3.14, -0.5, 2.0
bool Логический тип True, False

3. Ввод данных: input()

Функция input() позволяет получать данные от пользователя через консоль.

input.py
# Запрос имени
name = input("Как вас зовут? ")
print("Привет, " + name + "!")

# Важно: input() всегда возвращает строку
age = input("Сколько вам лет? ")
print(type(age))  # Выведет:
Преобразование типов:
  • int() — преобразует в целое число
  • float() — преобразует в дробное число
  • str() — преобразует в строку

4. Практические примеры

Пример 1: Калькулятор возраста

age_calc.py
# Запрашиваем год рождения
birth_year = int(input("Введите год рождения: "))

# Считаем возраст
current_year = 2026
age = current_year - birth_year

print(f"Вам {age} лет")

Пример 2: Конвертер валют

converter.py
# Курс доллара
usd_rate = 92.5

# Запрашиваем сумму в рублях
rubles = float(input("Сумма в рублях: "))

# Конвертируем
dollars = rubles / usd_rate

print(f"{rubles} рублей = {dollars:.2f} долларов")

Пример 3: Анкета пользователя

profile.py
# Сбор данных
name = input("Имя: ")
city = input("Город: ")
hobby = input("Хобби: ")

# Вывод анкеты
print("\n--- Профиль ---")
print(f"Имя: {name}")
print(f"Город: {city}")
print(f"Хобби: {hobby}")
Форматированный вывод: Используй f-строки для удобного вывода: print(f"Привет, {name}!"). Для float можно указать точность: {value:.2f} покажет 2 знака после запятой.
Практическое задание
Задача 1: Создай программу, которая запрашивает два числа и выводит их сумму, разность, произведение и частное.

Задача 2: Напиши программу "Паспорт", которая запрашивает имя, фамилию, возраст и город, затем выводит красивую анкету.
Подсказка: для преобразования ввода используй int() или float(). Для красивого вывода используй f-строки с переносом строки \n.
