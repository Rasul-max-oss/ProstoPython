Условные операторы

Конструкция if-elif-else позволяет программе принимать решения и выполнять разный код в зависимости от условий.

1. Конструкция if-else

Оператор if проверяет условие и выполняет код, если условие истинно. else выполняет альтернативный код, если условие ложно.

basic_if.py
age = 18

if age >= 18:
    print("Доступ разрешён")
else:
    print("Доступ запрещён")

print("Код продолжается...")
Важно: После if и else ставится двоеточие, а код внутри блока выполняется с отступом (4 пробела).

2. Операторы сравнения

Оператор Значение Пример
== Равно 5 == 5 → True
!= Не равно 5 != 3 → True
> Больше 5 > 3 → True
< Меньше 3 < 5 → True
>= Больше или равно 5 >= 5 → True
<= Меньше или равно 3 <= 5 → True

3. Конструкция elif

elif (else if) позволяет проверить несколько условий последовательно.

grading.py
score = 85

if score >= 90:
    print("Отлично")
elif score >= 75:
    print("Хорошо")
elif score >= 60:
    print("Удовлетворительно")
else:
    print("Нужно подучить")

4. Логические операторы

AND — оба условия истинны

and_operator.py
age = 25
has_license = True

if age >= 18 and has_license:
    print("Можно водить машину")

OR — хотя бы одно условие истинно

or_operator.py
is_weekend = True
is_holiday = False

if is_weekend or is_holiday:
    print("Можно отдыхать")

NOT — инверсия условия

not_operator.py
is_raining = False

if not is_raining:
    print("Можно гулять")

5. Практические примеры

Пример 1

Калькулятор скидок

Расчёт скидки в зависимости от суммы покупки

total = 3500
is_member = True

# Определяем базовую скидку
if total >= 5000:
    discount = 20
elif total >= 3000:
    discount = 10
elif total >= 1000:
    discount = 5
else:
    discount = 0

# Дополнительная скидка для членов клуба
if is_member:
    discount += 5

final_price = total * (100 - discount) / 100
print(f"Скидка: {discount}%")
print(f"К оплате: {final_price} руб.")
Пример 2

Проверка пароля

Сравнение введённого пароля с правильным

correct_password = "secret123"
user_input = input("Введите пароль: ")

if user_input == correct_password:
    print("Доступ разрешён")
else:
    print("Неверный пароль")
Пример 3

Калькулятор ИМТ

Индекс массы тела с оценкой результата

weight = float(input("Вес (кг): "))
height = float(input("Рост (м): "))

bmi = weight / (height ** 2)

print(f"Ваш ИМТ: {bmi:.1f}")

if bmi < 18.5:
    print("Недостаточный вес")
elif bmi < 25:
    print("Нормальный вес")
elif bmi < 30:
    print("Избыточный вес")
else:
    print("Ожирение")
Цепочка сравнений: В Python можно писать условия как в математике: if 0 < x < 10: вместо if x > 0 and x < 10:
