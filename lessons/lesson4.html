Циклы for и while

Циклы позволяют выполнять код многократно. Изучим два типа циклов в Python и управление ими.

1. Цикл for

Цикл for используется для перебора элементов последовательности или повторения действий заданное количество раз.

basic_for.py
# Повторить 5 раз
for i in range(5):
    print(f"Повторение {i}")

# Выведет: 0, 1, 2, 3, 4
Функция range():
  • range(5) — числа от 0 до 4
  • range(2, 6) — числа от 2 до 5
  • range(0, 10, 2) — числа 0, 2, 4, 6, 8 (шаг 2)

2. Примеры использования for

Пример 1

Сумма чисел

Вычисление суммы от 1 до 100

total = 0
for num in range(1, 101):
    total += num
print(f"Сумма: {total}")
Пример 2

Перебор строки

Вывод каждой буквы текста

name = "Python"
for letter in name:
    print(letter)
Пример 3

Нумерованный список

Вывод списка с номерами

items = ["Хлеб", "Молоко", "Яйца"]
for i, item in enumerate(items, 1):
    print(f"{i}. {item}")

3. Цикл while

Цикл while повторяется, пока условие истинно. Используется, когда количество повторений неизвестно заранее.

basic_while.py
# Счёт до 5
count = 1
while count <= 5:
    print(f"Счёт: {count}")
    count += 1

# Выведет: 1, 2, 3, 4, 5
Важно: Обязательно изменяй переменную в условии цикла, иначе получится бесконечный цикл.

4. Примеры использования while

Пример 1

Проверка пароля

Запрос пароля до правильного ввода

password = ""
while password != "secret123":
    password = input("Введите пароль: ")
print("Доступ разрешён!")
Пример 2

Обратный отсчёт

Запуск таймера с паузами

import time

countdown = 10
while countdown > 0:
    print(countdown)
    time.sleep(1)
    countdown -= 1
print("Старт!")
Пример 3

Меню программы

Работа программы до выхода пользователя

running = True
while running:
    print("\n1. Запуск")
    print("2. Выход")
    choice = input("Выбор: ")
    
    if choice == "2":
        running = False
        print("Выход из программы")

5. Управление циклами

Оператор break

Прерывает выполнение цикла досрочно.

break_example.py
# Поиск числа
for i in range(10):
    if i == 5:
        break
    print(i)
# Выведет: 0, 1, 2, 3, 4

Оператор continue

Пропускает текущую итерацию и переходит к следующей.

continue_example.py
# Только чётные
for i in range(5):
    if i == 2:
        continue
    print(i)
# Выведет: 0, 1, 3, 4
Конструкция else в циклах: После цикла можно добавить else, который выполнится, если цикл завершился естественно (без break).
