Кортежи и множества

Кортежи — неизменяемые последовательности. Множества — коллекции уникальных элементов. Изучим их особенности и применение.

1. Кортежи (tuple)

Кортеж — это неизменяемый список. Создаётся с помощью круглых скобок. После создания элементы изменить нельзя.

tuple_basic.py
# Создание кортежа
point = (10, 20)
colors = ("красный", "зелёный", "синий")

# Доступ к элементам
print(point[0])  # 10
print(colors[1]) # зелёный

# Кортеж из одного элемента
single = (42,)  # Запятая обязательна!
Зачем кортежи: Они быстрее списков, защищают данные от изменений, могут быть ключами в словарях.

2. Операции с кортежами

Пример 1

Распаковка кортежа

Присваивание элементов переменным

person = ("Иван", 25, "Москва")
name, age, city = person

print(f"{name}, {age} лет, из {city}")
Пример 2

Возврат нескольких значений

Функция возвращает кортеж

def get_min_max(numbers):
    return min(numbers), max(numbers)

result = get_min_max([5, 2, 8, 1, 9])
print(f"Мин: {result[0]}, Макс: {result[1]}")

3. Множества (set)

Множество — это неупорядоченная коллекция уникальных элементов. Создаётся с помощью фигурных скобок.

set_basic.py
# Создание множества
unique = {1, 2, 3, 3, 2}
print(unique)  # {1, 2, 3} - дубликаты удалены

# Пустое множество
empty = set()  # Не {} - это пустой словарь!

# Из списка во множество
numbers = [1, 2, 2, 3, 3, 3]
unique_numbers = set(numbers)
print(unique_numbers)  # {1, 2, 3}

4. Операции с множествами

ОперацияКодОписание
ОбъединениеA | BВсе элементы из A и B
ПересечениеA & BТолько общие элементы
РазностьA - BЭлементы A, которых нет в B
Симметричная разностьA ^ BЭлементы из A и B, кроме общих
set_operations.py
A = {1, 2, 3, 4}
B = {3, 4, 5, 6}

print(A | B)  # {1, 2, 3, 4, 5, 6} - объединение
print(A & B)  # {3, 4} - пересечение
print(A - B)  # {1, 2} - разность
print(A ^ B)  # {1, 2, 5, 6} - симметричная разность

5. Практические примеры

Пример 1

Удаление дубликатов

Очистка списка от повторяющихся элементов

emails = [
    "user@example.com",
    "admin@example.com", 
    "user@example.com",  # дубль
    "test@example.com",
    "admin@example.com"  # дубль
]

# Удаляем дубликаты
unique_emails = list(set(emails))
print(unique_emails)
Пример 2

Поиск общих друзей

Пересечение множеств

friends_ivan = {"Анна", "Борис", "Виктор", "Галина"}
friends_petr = {"Борис", "Виктор", "Дмитрий", "Елена"}

# Общие друзья
common = friends_ivan & friends_petr
print(f"Общие друзья: {common}")
Пример 3

Проверка уникальности

Проверка пароля на уникальные символы

password = "abc123"
unique_chars = len(set(password))

if unique_chars >= 6:
    print("Пароль достаточно уникальный")
else:
    print("Много повторяющихся символов")
Методы множеств: add() добавляет элемент, discard() удаляет (без ошибки если нет), clear() очищает множество.
