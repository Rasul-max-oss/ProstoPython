Словари

Словари хранят данные в виде пар ключ-значение. Это мощный инструмент для структурирования информации.

1. Создание словарей

Словарь создаётся с помощью фигурных скобок. Каждый элемент — это пара ключ:значение.

dict_basic.py
# Пустой словарь
empty = {}

# Словарь с данными
person = {
    "name": "Алекс",
    "age": 25,
    "city": "Москва"
}

# Доступ к значению по ключу
print(person["name"])  # Алекс
print(person["age"])   # 25

2. Основные операции

ОперацияКодОписание
Добавить/изменитьdict[key] = valueДобавляет или обновляет
Получить значениеdict.get(key)Безопасное получение
Удалитьdel dict[key]Удаляет по ключу
Все ключиdict.keys()Возвращает ключи
Все значенияdict.values()Возвращает значения
Все парыdict.items()Возвращает пары (ключ, значение)

3. Перебор словарей

dict_loop.py
person = {"name": "Алекс", "age": 25, "city": "Москва"}

# Перебор ключей
for key in person:
    print(key)

# Перебор значений
for value in person.values():
    print(value)

# Перебор пар ключ-значение
for key, value in person.items():
    print(f"{key}: {value}")

4. Практические примеры

Пример 1

База данных сотрудников

Хранение информации о сотрудниках

employees = {
    "Иванов": {"position": "Менеджер", "salary": 80000},
    "Петров": {"position": "Разработчик", "salary": 120000},
    "Сидоров": {"position": "Дизайнер", "salary": 90000}
}

# Доступ к данным
print(employees["Иванов"]["position"])  # Менеджер
print(employees["Петров"]["salary"])    # 120000
Пример 2

Подсчёт слов

Частота слов в тексте

text = "кот собака кот мышь собака кот"
words = text.split()

# Подсчёт частоты
frequency = {}
for word in words:
    frequency[word] = frequency.get(word, 0) + 1

print(frequency)
# {'кот': 3, 'собака': 2, 'мышь': 1}
Пример 3

Конвертер валют

Хранение курсов валют

rates = {
    "USD": 92.5,
    "EUR": 99.8,
    "CNY": 12.7,
    "GBP": 115.3
}

# Конвертация
rubles = 10000
currency = "USD"
result = rubles / rates[currency]
print(f"{rubles} руб. = {result:.2f} {currency}")

5. Метод get() для безопасного доступа

Метод get() возвращает None или значение по умолчанию, если ключ не найден.

safe_access.py
person = {"name": "Алекс", "age": 25}

# Без get() - ошибка если ключа нет
# print(person["email"])  # KeyError!

# С get() - безопасно
email = person.get("email", "не указан")
print(email)  # не указан

age = person.get("age", 0)
print(age)  # 25
Вложенные словари: Словари могут содержать другие словари и списки. Это позволяет создавать сложные структуры данных для хранения информации.
