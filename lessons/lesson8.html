Функции

Функции позволяют группировать код для повторного использования. Изучим создание функций, параметры и возврат значений.

1. Создание функций

Функция определяется ключевым словом def. После имени указываются параметры в скобках.

function_basic.py
# Определение функции
def greet():
    print("Привет!")
    print("Добро пожаловать")

# Вызов функции
greet()
greet()  # Можно вызывать многократно

2. Параметры и аргументы

Функции могут принимать данные через параметры.

function_params.py
# Функция с параметром
def greet(name):
    print(f"Привет, {name}!")

greet("Алекс")    # Привет, Алекс!
greet("Мария")    # Привет, Мария!

# Функция с несколькими параметрами
def add(a, b):
    print(f"Сумма: {a + b}")

add(5, 3)    # Сумма: 8
add(10, 20)  # Сумма: 30

3. Возврат значения

Оператор return возвращает результат из функции.

function_return.py
# Функция с возвратом
def multiply(a, b):
    return a * b

result = multiply(5, 3)
print(result)  # 15

# Использование в выражениях
total = multiply(2, 3) + multiply(4, 5)
print(total)  # 26
Важно: Функция без return возвращает None. return также завершает выполнение функции.

4. Параметры по умолчанию

Параметрам можно задать значения по умолчанию.

default_params.py
def greet(name, greeting="Привет"):
    print(f"{greeting}, {name}!")

greet("Алекс")              # Привет, Алекс!
greet("Мария", "Добрый день")  # Добрый день, Мария!
Правило: Параметры по умолчанию должны идти после обязательных параметров.

5. Практические примеры

Пример 1

Калькулятор

Функции для математических операций

def add(a, b):
    return a + b

def subtract(a, b):
    return a - b

def multiply(a, b):
    return a * b

def divide(a, b):
    if b != 0:
        return a / b
    return "Ошибка: деление на 0"

# Использование
print(add(10, 5))      # 15
print(divide(10, 2))   # 5.0
print(divide(10, 0))   # Ошибка: деление на 0
Пример 2

Форматирование текста

Функции для работы со строками

def format_name(first, last):
    """Форматирует имя и фамилию"""
    return f"{first.title()} {last.title()}"

def create_greeting(name, age):
    """Создаёт приветственное сообщение"""
    return f"Привет, {name}! Вам {age} лет."

# Использование
full_name = format_name("алекс", "иванов")
print(full_name)  # Алекс Иванов

message = create_greeting("Алекс", 25)
print(message)  # Привет, Алекс! Вам 25 лет.
Пример 3

Проверка данных

Функции валидации

def is_adult(age):
    """Проверяет совершеннолетие"""
    return age >= 18

def is_valid_password(password):
    """Проверяет надёжность пароля"""
    if len(password) < 8:
        return False
    if not any(c.isdigit() for c in password):
        return False
    return True

# Использование
print(is_adult(20))        # True
print(is_adult(15))        # False

print(is_valid_password("abc12345"))  # True
print(is_valid_password("abc"))       # False

6. Область видимости

Переменные внутри функции локальны и не видны снаружи.

scope.py
x = 10  # Глобальная переменная

def my_func():
    y = 5  # Локальная переменная
    print(x)  # Можно использовать глобальную
    print(y)  # Можно использовать локальную

my_func()
print(x)  # Можно
# print(y)  # Ошибка! y не видна снаружи
Документирование: Добавляй строку документации (docstring) сразу после def для описания функции. Это поможет понять назначение функции.
