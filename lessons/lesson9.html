Работа с файлами

Научимся читать и записывать файлы. Это необходимо для сохранения данных между запусками программы.

1. Открытие файлов

Для работы с файлами используется функция open(). Важно закрыть файл после работы.

file_open.py
# Открытие файла
file = open("data.txt", "r", encoding="utf-8")

# Чтение содержимого
content = file.read()
print(content)

# Закрытие файла
file.close()
Режимы открытия:
  • "r" — чтение (по умолчанию)
  • "w" — запись (перезаписывает файл)
  • "a" — добавление в конец
  • "x" — создание нового файла

2. Конструкция with

Конструкция with автоматически закрывает файл, даже если произошла ошибка.

file_with.py
# Рекомендуемый способ
with open("data.txt", "r", encoding="utf-8") as file:
    content = file.read()
    print(content)
# Файл автоматически закрыт

3. Чтение файлов

Пример 1

Чтение всего файла

Метод read() читает всё содержимое

with open("text.txt", "r", encoding="utf-8") as f:
    content = f.read()
    print(content)
Пример 2

Чтение по строкам

Метод readline() читает одну строку

with open("text.txt", "r", encoding="utf-8") as f:
    line1 = f.readline()
    line2 = f.readline()
    print(line1)
    print(line2)
Пример 3

Чтение всех строк

Метод readlines() возвращает список строк

with open("text.txt", "r", encoding="utf-8") as f:
    lines = f.readlines()
    for line in lines:
        print(line.strip())  # strip() удаляет \n

4. Запись в файлы

file_write.py
# Запись в файл (перезапись)
with open("output.txt", "w", encoding="utf-8") as f:
    f.write("Привет, мир!\n")
    f.write("Это мой первый файл.\n")

# Добавление в файл
with open("output.txt", "a", encoding="utf-8") as f:
    f.write("Новая строка\n")
Важно: При записи в режиме "w" старое содержимое файла удаляется. Используй "a" для добавления.

5. Практические примеры

Пример 1

Чтение заметок

Программа читает файл с заметками

def read_notes():
    try:
        with open("notes.txt", "r", encoding="utf-8") as f:
            notes = f.read()
            print("Мои заметки:")
            print(notes)
    except FileNotFoundError:
        print("Файл с заметками не найден")

read_notes()
Пример 2

Дневник

Запись мыслей в файл с датой

from datetime import datetime

def write_diary():
    thought = input("О чём думаешь? ")
    date = datetime.now().strftime("%Y-%m-%d %H:%M")
    
    with open("diary.txt", "a", encoding="utf-8") as f:
        f.write(f"[{date}] {thought}\n")
    
    print("Записано в дневник!")

write_diary()
Пример 3

Копирование файла

Чтение одного файла и запись в другой

def copy_file(source, destination):
    with open(source, "r", encoding="utf-8") as src:
        content = src.read()
    
    with open(destination, "w", encoding="utf-8") as dst:
        dst.write(content)
    
    print(f"Файл скопирован из {source} в {destination}")

copy_file("original.txt", "copy.txt")

6. Обработка ошибок

При работе с файлами могут возникать ошибки. Используй try-except для их обработки.

file_errors.py
try:
    with open("missing.txt", "r") as f:
        print(f.read())
except FileNotFoundError:
    print("Файл не найден!")
except PermissionError:
    print("Нет доступа к файлу!")
except Exception as e:
    print(f"Ошибка: {e}")
Кодировка: Всегда указывай encoding="utf-8" при работе с русским текстом, чтобы избежать проблем с кодировкой.
