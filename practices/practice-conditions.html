Задача 1: Положительное или отрицательноеЛёгкая

Запросите число и выведите, положительное оно, отрицательное или равно нулю.

Ввод:-5
Вывод:Отрицательное число

    print("Положительное число")

    print("Равно нулю")

Задача 8: КалькуляторСредняя

Ввод:10\n5\n+
Вывод:15

b = float(input("Число 2: "))

    print(a + b)

Задача 10: Скидка в магазинеСредняя

Запросите сумму покупки. Если больше 5000 — скидка 10%, если больше 10000 — 20%.

Ввод:7000
Вывод:Скидка: 700₽\nИтого: 6300₽
amount = float(input("Сумма: "))
if amount > 10000:
    discount = amount * 0.2
elif amount > 5000:
    discount = amount * 0.1
else:
    discount = 0
print(f"Скидка: {discount}₽")
print(f"Итого: {amount - discount}₽")
Задача 11: Координатная четвертьСложная

Запросите координаты X и Y. Определите четверть.

Ввод:5\n-3
Вывод:IV четверть
x = int(input("X: "))
y = int(input("Y: "))

    print("I четверть")
elif x < 0 and y > 0:
    print("II четверть")
elif x < 0 and y < 0:
    print("III четверть")
elif x > 0 and y < 0:
    print("IV четверть")
else:
    print("На оси")
Задача 12: БанкоматСложная

Запросите баланс и сумму снятия. Проверьте, хватает ли средств.

Ввод:5000\n2000
Вывод:Снято 2000₽\nОстаток: 3000₽
balance = float(input("Баланс: "))
withdraw = float(input("Снять: "))
if withdraw > balance:
    print("Недостаточно средств")
elif withdraw <= 0:
    print("Некорректная сумма")
else:
    balance -= withdraw
    print(f"Снято {withdraw}₽")
    print(f"Остаток: {balance}₽")
