📌 Уровень 1: Основы словарей

Задача 1: Создание словаряЛёгкая

Создайте словарь с информацией о себе (имя, возраст, город) и выведите его.

Вывод:{'name': 'Алекс', 'age': 25, 'city': 'Москва'}
person = {'name': 'Алекс', 'age': 25, 'city': 'Москва'}
print(person)
Задача 2: Доступ к значениюЛёгкая

Выведите значение по ключу 'name' из словаря.

Словарь:{'name': 'Мария', 'age': 30}
Вывод:Мария
person = {'name': 'Мария', 'age': 30}
print(person['name'])
Задача 3: Добавление элементаЛёгкая

Добавьте в словарь новый ключ 'email' со значением.

Словарь:{'name': 'Олег'}
Вывод:{'name': 'Олег', 'email': 'oleg@mail.ru'}
person = {'name': 'Олег'}
person['email'] = 'oleg@mail.ru'
print(person)
Задача 4: Перебор словаряСредняя

Выведите все ключи и значения словаря в формате "ключ: значение".

Словарь:{'a': 1, 'b': 2, 'c': 3}
Вывод:a: 1 b: 2 c: 3
data = {'a': 1, 'b': 2, 'c': 3}
for key, value in data.items():
    print(f"{key}: {value}")

⌨️ Уровень 2: Методы словарей

Задача 5: Метод getЛёгкая

Получите значение по ключу 'phone', если его нет — верните "Не указан".

Словарь:{'name': 'Иван'}
Вывод:Не указан
person = {'name': 'Иван'}
print(person.get('phone', 'Не указан'))
Задача 6: Удаление элементаЛёгкая

Удалите ключ 'age' из словаря.

Словарь:{'name': 'Анна', 'age': 25, 'city': 'СПб'}
Вывод:{'name': 'Анна', 'city': 'СПб'}
person = {'name': 'Анна', 'age': 25, 'city': 'СПб'}
del person['age']
print(person)
Задача 7: Подсчёт словСредняя

Посчитайте, сколько раз каждое слово встречается в списке.

Список:['яблоко', 'груша', 'яблоко', 'апельсин', 'груша', 'яблоко']
Вывод:{'яблоко': 3, 'груша': 2, 'апельсин': 1}
words = ['яблоко', 'груша', 'яблоко', 'апельсин', 'груша', 'яблоко']
count = {}
for word in words:
    count[word] = count.get(word, 0) + 1
print(count)
Задача 8: Телефонная книгаСредняя

Создайте словарь с контактами. Реализуйте поиск по имени.

Ввод:Алекс
Вывод:+7-999-123-45-67
phonebook = {
    'Алекс': '+7-999-123-45-67',
    'Мария': '+7-999-111-22-33',
    'Олег': '+7-999-444-55-66'
}
name = input("Имя: ")
print(phonebook.get(name, 'Контакт не найден'))

Задача 10: Пересечение множествСредняя

Найдите общие элементы двух множеств.

Множества:{1, 2, 3, 4} {3, 4, 5, 6}
Вывод:{3, 4}
set1 = {1, 2, 3, 4}

Задача 11: Объединение множествЛёгкая

Объедините два множества.

Вывод:{1, 2, 3, 4, 5, 6}

Задача 12: Уникальные слова в текстеСложная

Посчитайте количество уникальных слов в тексте (без учёта регистра).

Ввод:Python python Java JAVA python
Вывод:2 уникальных слова
text = input("Текст: ")
words = text.lower().split()
unique = set(words)
print(f"{len(unique)} уникальных слов")
