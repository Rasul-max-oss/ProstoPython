Задача 10: ЛоггерСложная

Создайте функцию log(message), которая записывает сообщение с датой и временем в файл 'log.txt'.

Ввод:log("Пользователь вошёл")
Файл:[2026-03-26 14:30:00] Пользователь вошёл
from datetime import datetime

def log(message):
    timestamp = datetime.now().strftime("%Y-%m-%d %H:%M:%S")
    with open('log.txt', 'a', encoding='utf-8') as f:
        f.write(f"[{timestamp}] {message}\n")

log("Пользователь вошёл")
