Задача 9: Простое числоСложная

Запросите число и проверьте, простое ли оно.

Ввод:17
Вывод:Простое число
n = int(input("Число: "))
is_prime = True
for i in range(2, int(n**0.5) + 1):
    if n % i == 0:
        is_prime = False
        break
if is_prime and n > 1:
    print("Простое число")
else:
    print("Не простое число")
Задача 10: Угадай числоСредняя

Компьютер загадывает число от 1 до 100. Угадывайте с подсказками.

Ввод:50 25 37
Вывод:Больше Меньше Угадал!
import random
secret = random.randint(1, 100)
while True:
    guess = int(input("Ваше число: "))
    if guess < secret:
        print("Больше")
    elif guess > secret:
        print("Меньше")
    else:
        print("Угадал!")
        break

Задача 13: Максимум в последовательностиСредняя

Пользователь вводит числа до 0. Найдите максимум.

Ввод:5 12 3 0
Вывод:Максимум: 12
max_num = float('-inf')
while True:
    num = int(input("Число (0 для выхода): "))
    if num == 0:
        break
    if num > max_num:
        max_num = num
print(f"Максимум: {max_num}")
Задача 14: Совершенное числоСложная

Проверьте, совершенное ли число (сумма делителей = числу).

Ввод:6
Вывод:Совершенное число
n = int(input("Число: "))
div_sum = 0
for i in range(1, n):
    if n % i == 0:
        div_sum += i
if div_sum == n:
    print("Совершенное число")
else:
    print("Не совершенное число")
