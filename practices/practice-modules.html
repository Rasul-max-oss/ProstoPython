Задача 2: Случайный элементЛёгкая

Задача 3: ПеремешиваниеЛёгкая

Задача 11: Разница датСредняя

Найдите количество дней между двумя датами.

Даты:2026-01-01 2026-03-26
Вывод:84 дней
from datetime import date
d1 = date(2026, 1, 1)
d2 = date(2026, 3, 26)
delta = d2 - d1
print(f"{delta.days} дней")
Задача 12: Секунды до полуночиСложная

Вычислите, сколько секунд осталось до полуночи.

Вывод:33420 секунд
from datetime import datetime, timedelta
now = datetime.now()
midnight = datetime(now.year, now.month, now.day) + timedelta(days=1)
delta = midnight - now
print(f"{int(delta.total_seconds())} секунд")
