Задача 12: Секунды до полуночиСложная

Вычислите, сколько секунд осталось до полуночи.

Вывод:33420 секунд
from datetime import datetime, timedelta
now = datetime.now()
midnight = datetime(now.year, now.month, now.day) + timedelta(days=1)
delta = midnight - now
print(f"{int(delta.total_seconds())} секунд")

📦 Уровень 5: Модуль json

Задача 18: Создание JSONЛёгкая

Создайте JSON-строку из словаря.

Словарь:{'name': 'Алекс', 'age': 25}
Вывод:{"name": "Алекс", "age": 25}
import json
data = {'name': 'Алекс', 'age': 25}
json_str = json.dumps(data, ensure_ascii=False)
print(json_str)
Задача 19: Парсинг JSONЛёгкая

Распарсите JSON-строку в словарь.

JSON:{"name": "Мария", "age": 30}
Вывод:Мария, 30 лет
import json
json_str = '{"name": "Мария", "age": 30}'
data = json.loads(json_str)
print(f"{data['name']}, {data['age']} лет")
Задача 20: Запись в JSON-файлСредняя

Запишите словарь в файл 'data.json'.

Словарь:{'users': [{'name': 'Алекс'}, {'name': 'Мария'}]}
Файл:data.json
import json
data = {'users': [{'name': 'Алекс'}, {'name': 'Мария'}]}
with open('data.json', 'w', encoding='utf-8') as f:
    json.dump(data, f, ensure_ascii=False, indent=2)
Задача 21: Чтение JSON-файлаСредняя

Прочитайте данные из файла 'data.json'.

Файл:{"count": 5, "items": [1, 2, 3]}
Вывод:count: 5, items: [1, 2, 3]
import json
with open('data.json', 'r', encoding='utf-8') as f:
    data = json.load(f)
print(f"count: {data['count']}, items: {data['items']}")

🧠 Уровень 6: Модуль collections

Задача 22: Counter - подсчётСредняя

Посчитайте количество каждого элемента в списке.

Список:['яблоко', 'банан', 'яблоко', 'апельсин', 'банан', 'яблоко']
Вывод:Counter({'яблоко': 3, 'банан': 2, 'апельсин': 1})
from collections import Counter
fruits = ['яблоко', 'банан', 'яблоко', 'апельсин', 'банан', 'яблоко']

Задача 23: defaultdictСредняя

Создайте словарь со значениями по умолчанию (списки).

Вывод:{'fruits': ['яблоко', 'банан'], 'vegetables': ['огурец']}
from collections import defaultdict
d = defaultdict(list)
d['fruits'].append('яблоко')
d['fruits'].append('банан')
d['vegetables'].append('огурец')
print(dict(d))
Задача 24: namedtupleСложная

Создайте именованный кортеж Person с полями name, age, city.

Вывод:Person(name='Алекс', age=25, city='Москва')
