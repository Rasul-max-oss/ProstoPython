Переменные и типы данных

15 задач для отработки создания переменных, ввода данных и преобразования типов

📌 Уровень 1: Основы

Задача 1: Приветствие Лёгкая

Создайте переменную name со своим именем и выведите сообщение "Привет, [имя]!".

Вывод: Привет, Алекс! 
name = "Алекс"
print(f"Привет, {name}!")
Задача 2: Информация о себе Лёгкая

Создайте три переменные: name, age, city. Выведите: "[Имя], [возраст] лет, из [город]".

Вывод: Мария, 22 лет, из Москва 
name = "Мария"
age = 22
city = "Москва"
print(f"{name}, {age} лет, из {city}")
Задача 3: Простая математика Лёгкая

Создайте переменные a = 15 и b = 7. Выведите сумму, разность, произведение и частное.

Вывод: Сумма: 22 Разность: 8 Произведение: 105 Частное: 2.14 
a = 15
b = 7
print(f"Сумма: {a + b}")
print(f"Разность: {a - b}")
print(f"Произведение: {a * b}")
print(f"Частное: {(a / b):.2f}")
Задача 4: Квадрат и куб числа Лёгкая

Создайте переменную num = 5. Выведите квадрат и куб этого числа.

Вывод: Квадрат: 25 Куб: 125 
num = 5
print(f"Квадрат: {num ** 2}")
print(f"Куб: {num ** 3}")
Задача 5: Обмен значений Средняя

Создайте x = 10 и y = 20. Поменяйте их значения местами без третьей переменной.

Вывод: x = 20, y = 10 
x = 10
y = 20
x, y = y, x
print(f"x = {x}, y = {y}")

⌨️ Уровень 2: Ввод данных (input)

Задача 6: Приветствие с вводом Лёгкая

Запросите имя пользователя и выведите приветственное сообщение.

Ввод: Дмитрий
Вывод: Привет, Дмитрий! Рад знакомству! 
name = input("Как вас зовут? ")
print(f"Привет, {name}! Рад знакомству!")
Задача 7: Возраст в следующем году Лёгкая

Запросите возраст и выведите, сколько лет будет в следующем году.

Ввод: 25
Вывод: В следующем году вам будет 26 лет 
age = int(input("Сколько вам лет? "))
print(f"В следующем году вам будет {age + 1} лет")
Задача 8: Конвертер валют Средняя

Запросите сумму в рублях и курс доллара. Выведите сумму в долларах.

Ввод: 10000 92.5
Вывод: 10000 рублей = 108.11 долларов 
rubles = float(input("Сумма в рублях: "))
rate = float(input("Курс доллара: "))
dollars = rubles / rate
print(f"{rubles} рублей = {dollars:.2f} долларов")
Задача 9: Среднее арифметическое Средняя

Запросите три числа и выведите их среднее арифметическое.

Ввод: 10 20 30
Вывод: Среднее: 20.0 
num1 = float(input("Число 1: "))
num2 = float(input("Число 2: "))
num3 = float(input("Число 3: "))
average = (num1 + num2 + num3) / 3
print(f"Среднее: {average}")
Задача 10: Калькулятор ИМТ Средняя

Запросите вес (кг) и рост (м). Вычислите ИМТ: вес / рост².

Ввод: 70 1.75
Вывод: Ваш ИМТ: 22.86 
weight = float(input("Вес (кг): "))
height = float(input("Рост (м): "))
bmi = weight / (height ** 2)
print(f"Ваш ИМТ: {bmi:.2f}")

🔥 Уровень 3: Сложные задачи

Задача 11: Время в секундах Средняя

Запросите часы, минуты, секунды. Переведите в секунды.

Ввод: 2 30 45
Вывод: Прошло секунд: 9045 
hours = int(input("Часы: "))
minutes = int(input("Минуты: "))
seconds = int(input("Секунды: "))
total = hours * 3600 + minutes * 60 + seconds
print(f"Прошло секунд: {total}")
Задача 12: Сдача в магазине Средняя

Запросите стоимость и сумму покупателя. Выведите сдачу.

Ввод: 350 500
Вывод: Сдача: 150 рублей 
price = float(input("Стоимость: "))
given = float(input("Дал: "))
change = given - price
print(f"Сдача: {change} рублей")
Задача 13: Прямоугольник Лёгкая

Запросите длину и ширину. Выведите периметр и площадь.

Ввод: 10 5
Вывод: Периметр: 30 Площадь: 50 
length = float(input("Длина: "))
width = float(input("Ширина: "))
perimeter = 2 * (length + width)
area = length * width
print(f"Периметр: {perimeter}")
print(f"Площадь: {area}")
Задача 14: Конвертер температур Средняя

Запросите температуру в °C. Переведите в °F: F = C × 9/5 + 32.

Ввод: 25
Вывод: 25°C = 77.0°F 
celsius = float(input("Температура (°C): "))
fahrenheit = celsius * 9/5 + 32
print(f"{celsius}°C = {fahrenheit}°F")
Задача 15: Анкета пользователя Сложная

Запросите: имя, фамилию, возраст, город, хобби. Выведите красивую анкету.

Ввод: Иван Петров 28 СПб Программирование
Вывод: ═══════════════════════════ АНКЕТА ПОЛЬЗОВАТЕЛЯ ═══════════════════════════ Имя: Иван Фамилия: Петров Возраст: 28 лет Город: СПб Хобби: Программирование ═══════════════════════════ 
line = "═" * 27
name = input("Имя: ")
surname = input("Фамилия: ")
age = input("Возраст: ")
city = input("Город: ")
hobby = input("Хобби: ")

print(line)
print("       АНКЕТА ПОЛЬЗОВАТЕЛЯ")
print(line)
print(f"Имя: {name}")
print(f"Фамилия: {surname}")
print(f"Возраст: {age} лет")
print(f"Город: {city}")
print(f"Хобби: {hobby}")
print(line)
